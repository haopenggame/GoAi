## 接口简介

即梦4.0是即梦同源的图像生成能力，该能力在统一框架内集成了文生图、图像编辑及多图组合生成的功能：支持单次输入最多 10 张图像及进行复合编辑，并能通过对提示词的深度推理，自动适配最优的图像比例尺寸与生成数量，可一次性输出最多 15 张内容关联的图像。此外，模型显著提升了中文生成的准确率与内容多样性，且支持 4K 超高清输出，为专业图像创作提供了从生成到编辑的一站式解决方案。

## 接入说明

### 限制条件

| 名称 | 内容 |
|------|------|
| **输入图要求** | 图片格式：仅支持JPEG、PNG格式，建议使用JPEG格式<br>图片文件大小：最大 15MB，支持最多输入10张图<br>图片分辨率：最大 4096 * 4096<br>输入图宽高比：（宽/高）范围：[1/3, 3] |
| **输出图说明** | 输出图详细宽高规则，参考下方width、height中参数描述<br>输出图会以列表形式返回，最大输出图数量 = 15 - 输入图数量 |
| **其他说明** | 输出图片的分辨率越大、输出图片数量越多、输入图片数量越多，延迟增加越明显<br>1次调用可能输出多张图片，根据输出图片张数进行计费，默认根据prompt理解意图判断输出图片数量<br>若对延迟&价格敏感，建议使用force_single参数，强制只输出1张图片 |

## 提交任务

### 提交任务请求参数

#### Query参数

拼接到url后的参数，示例：`https://visual.volcengineapi.com?Action=CVSync2AsyncSubmitTask&Version=2022-08-31`

| 参数 | 类型 | 可选/必选 | 说明 |
|------|------|------------|------|
| Action | string | 必选 | 接口名，取值：CVSync2AsyncSubmitTask |
| Version | string | 必选 | 版本号，取值：2022-08-31 |

#### Header参数

**注意**：本服务固定值：Region为cn-north-1，Service为cv

主要用于鉴权，详见公共参数 - 签名参数 - 在Header中的场景部分

#### Body参数

**注意**：业务请求参数，放到request.body中，MIME-Type为application/json

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| req_key | string | 必选 | 服务标识，取固定值: `jimeng_t2i_v40` |
| image_urls | array of string | 可选 | 图片文件URL，支持输入0至10张图<br>图片数量越多&图片尺寸越大，整体服务处理时间越长<br>输入图片建议控制在6张图片以内，图片过多，参考效果会降低 |
| prompt | string | 必选 | 用于生成图像的提示词，中英文输入均可<br>最长不超过800字符，prompt过长有概率出图异常或不生效<br>支持 prompt中直接指定生图比例，模型会根据"size"字段智能判断生图宽高比，默认生成2K分辨率、宽高比不超过3的图像<br>除""外不建议输入特殊的符号如$ |
| size | int | 可选 | 文生图场景：<br>面积和宽高需要2选1传入，都不传则默认面积取"2048*2048"，即生成2K分辨率图像且模型智能判断宽高比<br>面积和宽高同时输入时，优先使用宽高<br>只传面积，模型会根据用户prompt意图智能判断生图宽高比<br>图生图场景：<br>结合用户prompt意图、参考图尺寸，由模型智能判断生图宽高比<br><br>生成图片的面积，默认值：4194304，即2048*2048，生成2K分辨率图像<br>取值范围：[1024*1024, 4096*4096]，可生成1K到4K分辨率图像<br>建议生成2k以上的图片，过小分辨率图片较大概率出现人脸效果不佳、文字异常等效果问题 |
| width | int | 可选 | 1、生成图像宽高，默认该字段不传<br>2、宽高乘积在[1024*1024, 4096*4096]，且宽高比在[min_ratio, max_ratio]范围内<br>3、默认生成2K分辨率，宽高比不超过3的图像 |
| height | int | 可选 | 1、生成图像宽高，默认该字段不传<br>2、宽高乘积在[1024*1024, 4096*4096]，且宽高比在[min_ratio, max_ratio]范围内<br>3、默认生成2K分辨率，宽高比不超过3的图像 |
| return_url | int | 可选 | 是否返回可访问的URL，默认值：1<br>1：返回可访问的URL<br>0：不返回可访问的URL |
| force_single | bool | 可选 | 是否强制只输出1张图片，默认值：false<br>true：强制只输出1张图片<br>false：根据prompt意图判断输出图片数量 |
| mask_ids | array of string | 可选 | 蒙版图片的URL列表，支持0-10张<br>支持框选任意区域编辑，详细使用说明请参考蒙版编辑使用说明 |

## 提交任务响应参数

### 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| req_key | string | 请求对应的service identifier |
| request_id | string | 请求唯一ID，用于排查问题 |
| time_elapsed | string | 任务耗时 |
| status | int | 状态码 |

```json
    "status": 50215
}
```
